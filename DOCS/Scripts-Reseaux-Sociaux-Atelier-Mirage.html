02 · TikTok

Stratégie TikTok — ton deuxième levier

TikTok n'est pas "Instagram pour les jeunes". C'est une machine à viralité différente avec ses propres règles. Voici comment l'exploiter pour Atelier Mirage.

📸 Instagram — Ce qui reste pareil

  • Ta base de contenu — Tous les scripts ci-dessus marchent sur les deux plateformes
  • Ton audience B2B — Les restaurateurs et pâtissiers sont plus actifs sur Instagram
  • Le feed cohérent — Ton grid visuel bien tenu reste ta "carte de visite" pour les prospects
  • Les carrousels éducatifs — Format exclusif Instagram, très "saveable", parfait pour B2B
  • Les DM de prospection — Instagram DM reste le canal #1 pour le B2B food local
  • Stories interactives — Sondages, quiz, Q&A — Instagram est meilleur que TikTok pour ça

🎵 TikTok — Ce qui change

  • L'algo favorise les NOUVEAUX comptes — TikTok pousse tes premières vidéos plus qu'Instagram. C'est le moment de se lancer
  • Le son est ROI — Contrairement à Instagram (85% muet), TikTok = 70% avec son. L'ASMR et les sons trending sont essentiels
  • Les trends sont obligatoires — Chaque semaine, adapte un audio trending au trompe-l'œil. Un Reel "evergreen" sur Insta peut être un flop sur TikTok s'il n'est pas dans la trend
  • La durée idéale est plus courte — 7-15 secondes performe mieux que 20-30s sur TikTok. Tes reveals doivent être plus rapides
  • Les commentaires SONT le contenu — Réponds aux commentaires en vidéo ("Reply to @user"). C'est un format gratuit qui génère du contenu en boucle
  • Le texte en haut — Sur TikTok, les overlays doivent être dans la zone safe (pas trop bas, pas sur les boutons). Différent d'Instagram
  • Pas de "link in bio" aussi puissant — TikTok Shop existe mais c'est B2C. Pour B2B, redirige vers ton Instagram dans ta bio TikTok

La stratégie crosspost simplifiée : Filme UNE FOIS pour les deux plateformes. Monte DEUX versions : une version Instagram (15-30s, sous-titrée, musique trending Instagram) et une version TikTok (7-15s, plus punchy, son trending TikTok, texte en zone safe). C'est 10 min de montage supplémentaire pour doubler ton reach.

03 · Crosspost

Une session, deux plateformes

Tu n'as pas le temps de créer du contenu séparé pour chaque plateforme. Voici comment filmer une fois et publier partout intelligemment.

📹 Au tournage

Filme TOUJOURS en 9:16 (vertical). Pas de différence entre Insta et TikTok au tournage. Par contre, filme 2 versions de chaque intro : une version "hook parlé" (pour TikTok — tu parles à la caméra) et une version "texte overlay silencieux" (pour Instagram — le hook est écrit).

✂️ Au montage

Version Instagram (15-30s) : Sous-titres CapCut, musique via Instagram directement (meilleur pour l'algo), texte overlay centré, logo fin.
Version TikTok (7-15s) : Même rushes mais montage plus court, son trending TikTok, texte en zone safe (pas dans les 15% du bas), pas de watermark Instagram.

⏰ À la publication

Instagram : 18h (ton créneau peak). TikTok : 12h ou 19h-21h (les heures food performent le soir sur TikTok). Ne poste PAS au même moment — espace de 2-3h minimum. Et surtout : ne poste JAMAIS une vidéo avec le watermark TikTok sur Instagram (et vice versa). L'algo la pénalise.

🔄 Le recyclage intelligent

Un Reel Instagram qui marche bien (>500 vues) : Re-monte en version TikTok courte 1-2 jours plus tard.
Un TikTok qui explose (>5K vues) : Adapte en Reel Instagram avec sous-titres + boost Meta Ads.
Un carrousel Instagram : Adapte en vidéo "list" sur TikTok (voix off + texte).

Les 3 erreurs fatales du crosspost : 1) Poster une vidéo avec le logo TikTok sur Instagram → -30% de reach. 2) Utiliser la même musique trending sur les deux → elle ne trend pas forcément sur l'autre plateforme. 3) Copier-coller la même légende → les hashtags Instagram ne marchent pas sur TikTok (et inversement).
